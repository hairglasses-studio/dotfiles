# sway-mcp

> **Deprecated** -- This project has been consolidated into [dotfiles-mcp](https://github.com/hairglasses-studio/dotfiles-mcp).

All tools from this repo are now available as part of the unified `dotfiles-mcp` server (86 tools covering Hyprland, shaders, input devices, Bluetooth, MIDI, OSS scoring, and more).

## Migration

If you were using this server directly, switch to dotfiles-mcp:

```bash
go install github.com/hairglasses-studio/dotfiles-mcp@latest
```

Tools have been renamed with the `sway_` prefix in dotfiles-mcp (e.g., `sway_get_tree`).
