## Learning Path Suggestions

### Just Starting Out

1. **Week 1-2**: Git & GitHub basics (freeCodeCamp crash course)
2. **Week 3-4**: Python fundamentals (Programming with Mosh)
3. **Week 5-6**: Shell scripting basics (Traversy Media)
4. **Week 7+**: Pick your specialty (TD, Resolume, Bots, etc.)

### VJ/Visual Focus

1. Start with Resolume basics (DocOptic training)
2. Then TouchDesigner (elekktronaut series)
3. Add Python for automation
4. Learn OSC for control

### Automation Focus

1. Python basics (Corey Schafer)
2. Shell scripting (freeCodeCamp)
3. FFmpeg for video (Fireship)
4. Discord bots if needed

## Tips for Video Learning

1. **Code Along** - Don't just watch, type the code
2. **Pause Often** - Let concepts sink in
3. **Experiment** - Try variations of what's shown
4. **Take Notes** - Write key commands/concepts
5. **Practice Daily** - Even 30 minutes helps
6. **Build Projects** - Apply what you learn
